## Key Features
- **Dynamic Current Standings:** A real-time (at the time of creation) snapshot of ATP rankings and points.
- **Head-to-Head Deep Dives:** Comparative analysis of career trajectories for Federer, Nadal, and Djokovic.
- **Surface-Specific Metrics:** Logic-driven insights into win rates on Clay vs. Grass vs. Hard courts.
- **Longevity Analysis:** Visualization of match wins and titles per year across professional lifespans.

## Technical Implementation
- **Data Source:** Historical ATP Tour match statistics and rankings.
- **Modeling:** Built a relational schema to connect player profiles with match-level outcomes.
- **DAX Measures:** Developed custom expressions for:
  - *Win % Calculations* (Total wins / Total matches played).
  - *Title Trajectories* (Cumulative titles over career years).
